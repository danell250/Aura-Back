# Aura Social

This contains everything you need to run your app locally.

## Run Locally

**Prerequisites:**  Node.js

1. **Install dependencies:**
   ```bash
   npm install
   cd backend && npm install && cd ..
   ```

2. **Configure Environment:**
   - Create a `.env` file in the `backend` directory.
   - Add your Gemini API Key:
     ```env
     GEMINI_API_KEY=your_actual_api_key_here
     PORT=5002
     ```
   - Create a `.env` file in the `backend` directory.
     - Add your database and authentication configuration:
     ```env
     JWT_SECRET=superlongrandomsecret123!@#_doNotShare
     MONGO_URI=mongodb+srv://aurasocialradiate_db_user:oJYDjBMVBWIiQa5M@auracluster.getkwwb.mongodb.net/aura?retryWrites=true&w=majority
     PORT=5002
     GEMINI_API_KEY=your_actual_api_key_here
     ```
   - Create a `.env` file in the `frontend` directory with your API configuration:
     ```env
     # Backend and Frontend URLs
     VITE_BACKEND_URL=https://aura-back-s1bw.onrender.com
     VITE_API_BASE_URL=https://aura-back-s1bw.onrender.com
     VITE_FRONTEND_URL=https://auraraidiate.netlify.app/
     
     # Firebase Configuration
     VITE_FIREBASE_API_KEY=AIzaSyDmr4WusWj7jTOcx_h8dFqiNoGeDWRF-Sk
     VITE_FIREBASE_AUTH_DOMAIN=aura-2d406.firebaseapp.com
     VITE_FIREBASE_PROJECT_ID=aura-2d406
     VITE_FIREBASE_STORAGE_BUCKET=aura-2d406.firebasestorage.app
     VITE_FIREBASE_MESSAGING_SENDER_ID=788922070688
     VITE_FIREBASE_APP_ID=1:788922070688:web:59c8e18dd93be22fdb3bc5
     VITE_FIREBASE_MEASUREMENT_ID=G-K8VP4H8599
     ```
   - You can use the `.env.example` file in the frontend directory as a template.

3. **Run the app:**
   ```bash
   npm run dev
   ```
   This command will start both the backend server (port 5002) and the frontend development server (port 5000) concurrently.

## Production URLs

- **Frontend**: https://auraradiance.netlify.app/
- **Backend**: https://aura-back-s1bw.onrender.com

## Architecture

- **Frontend**: React + Vite (Port 5000)
- **Backend**: Express + TypeScript (Port 5002)
- **API Proxy**: Frontend requests to `/api/*` are proxied to the backend.

## Development

- Frontend code is in the root directory.
- Backend code is in the `backend` directory.
